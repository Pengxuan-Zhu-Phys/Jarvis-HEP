# Jarvis-HEP v1.5

Release date: March 1, 2026

`v1.5` is the final GitHub release baseline before PyPI publication, prepared for the paper release workflow.

## Highlights

- Removed the internal `JarvisPLOT` package from Jarvis-HEP to reduce coupling and simplify project scope.
- Stabilized the workflow around nuisance handling and profile-based usage patterns.
- Improved scanning/sampling workflows and examples used in practical studies.
- Continued cleanup and refinement of runtime behavior, configuration, and documentation.

## Changes Since `v1.3.0-beta`

- Added and updated sampling-related capabilities and examples across common workflows.
- Upgraded monitor and workflow visualization components.
- Improved HDF5/CSV output workflow and project usability details.
- Refined configuration and schema handling in core runtime paths.
- Performed substantial repository cleanup and documentation updates.

## Breaking / Behavior Notes

- Internal plotting module removal.
- The built-in `JarvisPLOT` module in this repository is no longer included.
- Plotting should be handled via external plotting tools/workflows.

## Compatibility Notes

- This release is intended as a stable, citable GitHub milestone prior to PyPI publishing.
- For paper reproducibility, pin your workflow to tag `v1.5`.
