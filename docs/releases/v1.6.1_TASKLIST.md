# Jarvis-HEP v1.6.1 Migration And Release Tasklist

Last updated: 2026-03-02  
Owner: Jarvis-HEP maintainers

## Scope Freeze (Task 1) - DONE

This release line is explicitly constrained to **entrypoint and packaging migration**.

- In scope:
  - CLI entry migration (`Jarvis` -> `jarvishep.client:main`)
  - Packaging/release wiring updates for PyPI
  - Compatibility-preserving import/path refactor work
  - Test and release verification improvements
- Out of scope:
  - Any change to sampling algorithms
  - Any change to likelihood/pass-condition calculation semantics
  - Any behavior change to workflow execution logic beyond import/entry wiring

Acceptance rule:

- No PR/change in this line may intentionally alter numerical physics results.
- Changes must be validated by current unit tests and packaging checks.
